Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for HTML, representing the page as a tree structure. It lets you manipulate elements, styles, and content dynamically.

How do you select an element from the DOM?

Use methods like document.getElementById('id'), document.querySelector('selector'), and document.querySelectorAll('selector') to select elements.

What is event delegation, and why is it useful?

Event delegation is adding a single event listener to a parent element to manage events for all its child elements. It reduces event listeners, boosts performance, and works with dynamic elements.

How do you manipulate an element's attributes and styles?

Use setAttribute/getAttribute for attributes and style.property or classList methods (like add, remove) for styles.
